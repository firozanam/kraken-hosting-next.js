# WHMCS API Integration

This section documents the WHMCS API integration for the Kraken Hosting platform.

## Overview

The Kraken Hosting application integrates with WHMCS through its API to provide a modern client and admin portal. The integration is handled through a centralized API wrapper that manages all communication with the WHMCS API.
